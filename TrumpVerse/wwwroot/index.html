TrumpStaffAPI

Key Type Description
id int The staffer's id
name string The staffer's name. Mandatory field.
age int The staffer's age. Optional.
position string The position the staffer has in the Trump system. Mandatory.
criminalCharges string Whether the staffer has any criminal charges against them. Optional.
convicted boolean Whether the staffer has been convicted of a crime.
image string (url) An image of the staffer.
